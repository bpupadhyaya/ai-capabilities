ONNX Runtime Web

Run ONNX-format models directly in the browser at near-native performance. Lower-level than Transformers.js — gives you direct control over execution providers, tensor shapes, and quantization.

⚙️ What is ONNX Runtime Web?

ONNX (Open Neural Network Exchange) is an open format for representing machine learning models. ONNX Runtime Web executes them in the browser — supporting both WebAssembly (CPU, works everywhere) and WebGPU (GPU accelerated, Chrome 113+) backends. Unlike Transformers.js which wraps models in a high-level pipeline API, ONNX Runtime Web gives you direct access to tensor I/O, letting you load any .onnx file and run inference.
🏃 WASM Backend Universal
  • ✓ Works in all modern browsers
  • ✓ iOS and Android supported
  • ✓ Multi-threaded via SharedArrayBuffer
  • ~ Slower than GPU (5-20x for large models)
  • ✓ Best for small models <100MB
⚡ WebGPU Backend GPU Accelerated
  • ✓ Hardware GPU acceleration
  • ✓ 10-50x faster for large models
  • ✗ Chrome 113+ / Edge 113+ only
  • ~ Safari 17+ macOS only
  • ✓ Best for 1B+ parameter models

💻 How to Use ONNX Runtime Web

<!-- Load from CDN -->
<script src="https://cdn.jsdelivr.net/npm/onnxruntime-web@1.20.0/dist/ort.min.js"></script>
<!-- For WebGPU support -->
<script src="https://cdn.jsdelivr.net/npm/onnxruntime-web@1.20.0/dist/ort.webgpu.min.js"></script>

// Configure WASM paths
ort.env.wasm.wasmPaths = 'https://cdn.jsdelivr.net/npm/onnxruntime-web@1.20.0/dist/';

// Load a model (fetch as ArrayBuffer)
const modelBuffer = await (await fetch('model.onnx')).arrayBuffer();

// Create session — WebGPU with WASM fallback
let session;
try {
  session = await ort.InferenceSession.create(modelBuffer, {
    executionProviders: ['webgpu'],
    graphOptimizationLevel: 'all',
  });
} catch {
  session = await ort.InferenceSession.create(modelBuffer, {
    executionProviders: ['wasm'],
  });
}

// Inspect model I/O
console.log('Inputs:',  session.inputNames);   // e.g. ['input_ids', 'attention_mask']
console.log('Outputs:', session.outputNames);  // e.g. ['logits']

// Run inference
const inputIds = new BigInt64Array([101n, 7592n, 2088n, 102n]);
const feeds = {
  input_ids:      new ort.Tensor('int64', inputIds, [1, 4]),
  attention_mask: new ort.Tensor('int64', new BigInt64Array([1n, 1n, 1n, 1n]), [1, 4]),
};

const results = await session.run(feeds);
const logits = results['logits'];
console.log('Output shape:', logits.dims);   // [1, 4, vocab_size]
console.log('Output data:',  logits.data);   // Float32Array
