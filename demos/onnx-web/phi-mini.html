Phi-3 Mini via ONNX Runtime Web

This demo runs Microsoft's Phi-3 Mini model using ONNX Runtime Web — the same runtime that powers on-device AI in Windows and Azure. Select an execution provider and click Load.

Loading approach:
  1. Select provider: WebGPU (fast, Chrome 113+) or WASM (slower, universal)
  2. Click Load — model downloads from Hugging Face CDN (~2.2GB, cached after first load)
  3. Uses Xenova/Phi-3-mini-4k-instruct via Transformers.js ONNX pipeline
  4. For direct ONNX loading, a server is needed due to model sharding (see ORT Web repo)
