Transformers.js Demos

Run Hugging Face models directly in your browser via ONNX Runtime WebAssembly. No WebGPU required — works in any modern browser, fully offline after first download.

WebAssembly ONNX Runtime Optional WebGPU

⚙️ How It Works

Transformers.js

  • Port of Hugging Face Transformers library to JavaScript
  • Uses ONNX Runtime Web for model execution
  • Falls back to CPU/WASM — works in any modern browser
  • Models downloaded from Hugging Face CDN, cached in browser
  • Supports 1000+ tasks: NLP, vision, audio, multimodal
  • Easy pipeline() API — identical to Python transformers

vs. WebLLM (WebGPU)

  • WebLLM uses WebGPU for GPU-accelerated generation (much faster)
  • Transformers.js runs on CPU/WASM — slower but universally compatible
  • Transformers.js handles more task types (vision, audio, classification)
  • WebLLM focuses on chat/generation with larger LLMs
  • Both are fully private — no data leaves the browser
  • Choose based on browser support needs and task type

💻 Usage

// Load Transformers.js via CDN — no build step needed
import { pipeline, env } from
  'https://cdn.jsdelivr.net/npm/@xenova/transformers@2.17.2/dist/transformers.min.js';

// Disable local model lookup (use HF CDN)
env.allowLocalModels = false;

// Create a sentiment analysis pipeline — model downloads on first use
const classifier = await pipeline(
  'sentiment-analysis',
  'Xenova/distilbert-base-uncased-finetuned-sst-2-english'
);

const result = await classifier("I love running AI in the browser!");
// → [{ label: 'POSITIVE', score: 0.9998 }]

// Text generation
const generator = await pipeline('text-generation', 'Xenova/distilgpt2');
const out = await generator("The future of AI is", { max_new_tokens: 50 });
// → [{ generated_text: "The future of AI is ..." }]

// Feature extraction (embeddings)
const extractor = await pipeline('feature-extraction', 'Xenova/all-MiniLM-L6-v2');
const embedding = await extractor("Hello world", { pooling: 'mean', normalize: true });
// → Float32Array of 384 dimensions
