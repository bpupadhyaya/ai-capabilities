Hardware Tier Guide — What Can You Run?

RAM Tier VRAM (GPU) Recommended Models Expected Speed Notes
8 GB RAM 4–6 GB VRAM Llama 3.2 3B Q4, Phi-4 Mini Q4, Gemma 3 4B Q4, Mistral 7B Q4 (tight) 15–40 tok/s (CPU) Close all other apps. Use Q4_K_M quantization. GPU layers help if VRAM available.
16 GB RAM 8–12 GB VRAM Llama 3.1 8B, Gemma 3 12B Q4, Phi-4 Q4, Mistral 7B F16, Qwen2.5 7B 20–60 tok/s Sweet spot for most users. Excellent Apple Silicon performance at this tier.
32 GB RAM 16–24 GB VRAM Gemma 3 27B Q4, Qwen2.5-Coder 32B Q4, Aya Expanse 32B Q4, DeepSeek-R1-Distill 7B F16 15–35 tok/s Covers most use cases including code generation. M2/M3 Pro/Max Macs excel here.
64 GB RAM 40–48 GB VRAM Llama 3.3 70B Q4, Qwen2.5 72B Q4, Mixtral 8x7B Q8, DeepSeek-R1-Distill 70B Q4 8–20 tok/s Mac Studio/Pro M2 Ultra/M3 Ultra class. Workstation territory. Outstanding capability.
128 GB+ RAM 80 GB+ VRAM DeepSeek R1 (671B) Q4, Qwen3 235B-A22B Q4, Llama 3.3 70B F16 + other models simultaneously 5–15 tok/s High-end workstations, Mac Pro, or multi-GPU setups. Frontier open-source quality.

Inference Engines for Desktop

🦙
Ollama
One-command install. ollama run llama3.3 and you're chatting. Model library, REST API, GPU offloading built in. Best beginner experience. ollama.ai
🎛
LM Studio
Desktop GUI. Browse, download, and chat with any GGUF model. Local OpenAI-compatible server. Ideal for non-technical users and model comparison. lmstudio.ai
⚙
llama.cpp
The underlying engine for most tools. C++ binary, CPU + GPU offloading, GGUF format. Advanced users tune -ngl GPU layers, context size, and threading. GitHub
📦
GPT4All
User-friendly desktop app with built-in document Q&A (LocalDocs). Cross-platform. Focused on privacy-preserving personal assistant use cases. gpt4all.io
🌙
Jan
Open-source ChatGPT alternative. Native desktop app with model hub, threads, and a local API server. Extensions support. Clean modern UI. jan.ai
🐲
KoboldCpp
Single-file llama.cpp with a rich web UI. Popular for creative writing and roleplay. Built-in story mode, prompt templates, KoboldAI client compatibility. GitHub
🍏
MLX (Apple Silicon)
Apple's machine learning framework. Uses unified memory for extremely fast inference on M-series Macs. mlx_lm.generate --model mlx-community/gemma-3-27b-it-4bit GitHub
🚄
vLLM
Production serving for NVIDIA GPUs. PagedAttention, continuous batching, high throughput. Best for local GPU servers shared by multiple users or applications. GitHub
