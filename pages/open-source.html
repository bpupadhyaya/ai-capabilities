Why Open Source?

🔒
Privacy by Default
Your prompts and data never leave your device. No vendor can read, log, or train on your conversations. True confidentiality for personal, legal, medical, and business use.
💰
Zero Cost at Inference
Once downloaded, inference is free. No per-token fees, no subscription, no rate limits. Run a million tokens a day without a billing alert.
🔧
Fully Customizable
Fine-tune on your own data with LoRA or full fine-tuning. Merge models. Change system prompts. Modify tokenizers. No vendor lock-in.
✈
Works Offline
Run in a Faraday cage, on a plane, in a basement, or in a country with internet censorship. No connectivity dependency for inference.

Get Started in 60 Seconds

Fastest path: Install Ollama and run ollama run llama3.2:3b for a fully local chat model on any laptop. For mobile, use MLC LLM for Android/iOS or MediaPipe for Gemma 3n on-device. For browser-only: try the WebLLM demo — zero install required.
